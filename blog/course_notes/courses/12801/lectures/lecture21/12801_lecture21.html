Rossby Waves Review

We have the following set-up for linear Rossby waves: \begin{align} \partial_b + wN^2 &= 0, \\ \partial_t q + \beta v &= 0, \end{align} where $b = f \partial_z\psi$, which gives: \begin{equation} q = \nabla^2\psi + \beta y + f_0^2 \partial_z\left[\frac{1}{N^2}\partial_z\psi\right]. \end{equation} Theb, we assume a solution of the form: \begin{equation} \psi = \psi_0\exp\left[\text{i}(kx + ly - \omega t)\right], \end{equation} and then we get, for $F = f_0^2 / N^2$ and $K^2 = k^2 + l^2$, we have: \begin{equation} K_m = m\pi\left(\frac{f_0}{ND} \right). \end{equation} The solution for the 1st baroclinic Rossby wave speed is then given by: \begin{equation} c_{R_1} = - \frac{\beta}{K^2 + \left( \pi f / N D \right)^2} \end{equation} We note that the wavelength is $2 \pi / L$ and we make the long-wave approximation $L \gg (N D / f)$. The doppler shifted Rossby Wave speed is given by: \begin{equation} u_d = u - c_{R_1}, \end{equation} and we can see this from linearizing conserved PV about a zonal mean (see Q1 and Q3 in 12.802.PS3)
