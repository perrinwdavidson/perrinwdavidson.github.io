12.802: Problem Set 3

Problem 1

(a) We want to derive an approximation for the potential vorticity of a 1-layer homogenous model on the $\beta$-plane with a constant depth $H$ in the absence of external forcing (i.e. it is conserved): \begin{equation} \frac{\textrm{D}}{\textrm{D}t}\left[\frac{f + \zeta}{h}\right] = 0, \end{equation} where we define the following terms: \begin{align} \zeta &= \frac{\partial v}{\partial x} - \frac{\partial u}{\partial y}, \\ f &= f_0 + \beta y, \\ h &= H + \eta, \end{align} and the derivative is the material derivative, defined as $\partial_t + \underline{u}\cdot\nabla$. To derive this approximation, we first note and suppose a few things: \begin{align} |\eta| &\ll |H|, \\ |\zeta| &\ll |f_0|, \\ |\beta y| &\ll |f_0|. \end{align} We further remember the following useful Taylor expansion: \begin{equation} \frac{1}{1 - x} = \sum_{n=0}^{\infty} x^n. \end{equation} So, we first can see: \begin{equation} \frac{f + \zeta}{h} = \frac{\zeta + f}{H}\frac{1}{\left(1 + \eta / H\right)}. \end{equation} Then, we can apply the Taylor expansion from above to the second fraction and only consider first order terms, which gives: \begin{equation} \frac{\zeta + f}{H}\left(1 - \frac{\eta}{H}\right) = \frac{1}{H}\left(\zeta + f - \zeta \frac{\eta}{H} - f_0\frac{\eta}{H} - \beta y \frac{\eta}{H}\right). \end{equation} By the suppositions above, we can reduce this to: \begin{equation} \frac{1}{H}\left(\zeta + f - f_0 \frac{\eta}{H}\right). \end{equation} As $1 / H$ and $f_0$ are constants, by plugging in the above equation to the material derivative, we can simplify this expression to: \begin{equation} \frac{\textrm{D}}{\textrm{D}t}\left[\zeta + \beta y - f_0 \frac{\eta}{H}\right] = 0. \end{equation}

(b) We now want to linearize the equation from above about a rest state, i.e. $\overline{\underline{u}} = \overline{\eta} = 0$ via perturbation quantities of order $\epsilon \ll 1$, i.e.: \begin{align} u\left(x, y, t\right) &= \epsilon u'\left(x, y, t\right), \\ v\left(x, y, t\right) &= \epsilon v'\left(x, y, t\right), \\ \eta\left(x, y, t\right) &= \epsilon \eta'\left(x, y, t\right), \end{align} where we let: \begin{align} u' &= - \frac{f_0}{g} \partial_y \eta, \\ v' &= \frac{f_0}{g} \partial_x \eta. \end{align} Plugging these perturbation quantities into the equation of interest, we get the following: \begin{align} \frac{\textrm{D}}{\textrm{D}t}\left[\zeta + \beta y - f_0 \frac{\eta}{H}\right] = \partial_t\left[\epsilon\partial_xv' - \epsilon\partial_yu' - \epsilon\frac{f_0}{H}\eta' \right] + \epsilon \beta v'\partial_yy =0, \end{align} where we have neglected all terms higher than $\epsilon$, and noting that $\partial_{x_i} x_j = \delta_{ij}$ (for $x_i,x_j$ coordinates $x,y$), we get the linearized equation: \begin{equation} \partial_{tx}v' - \partial_{ty}u' + \beta v' - \frac{f_0}{H} \partial_t \eta' = 0. \end{equation} Now, we can plug in our geostrophic equations for $u', v'$ giving: \begin{equation} \frac{g}{f_0} \partial_{tx x}\eta + \frac{g}{f_0}\partial_{t y y} \eta' + \frac{g}{f_0}\beta\partial_x\eta' - \frac{f_0}{H}\partial_t \eta' = 0. \end{equation} Noting that this is an equation only of sea surface anamoly $\eta'$, we assume a solution of the form $\eta' \sim \eta_0 \exp\left[i\left(kx+ly-\omega t\right)\right]$. Plugging this into our linear equation of $\eta'$ and solving for $\omega$ we get the dispersion relation: \begin{equation} \omega = - \frac{\beta k}{K^2 + k_d^2}, \end{equation} where $K^2 = k^2 + l^2$ is the horizontal wave number and $k_d^2 = \left(f_0^2 / gH\right)$ is the Rossby Deformation Radius. We note that this is the dispersion relation for Rossby waves with a finite deformation radius linearized around a rest state (i.e., $\overline{\underline{u}}=0.$)
