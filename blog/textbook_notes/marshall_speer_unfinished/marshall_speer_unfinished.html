We also note that there are two main coordinate systems that we use. The first is Cartesian Coordinates, of the form $(\widehat{\imath}, \widehat{\jmath}, \widehat{k})$: \begin{equation} \underline{x} = x \widehat{\imath} + y \widehat{\jmath} + z \widehat{k}, \end{equation} for the wide hat being the unit vector in each of the 3 dimensions. We also use Spherical Coordinates $(\widehat{\lambda}, \widehat{\phi}, \widehat{r})$, which are diagramatically presented in Fig. 1.
We claim that we can represent the coordinate $\underline{x}$ in spherical coordinates using the following increment relation: \begin{equation} \delta \underline{x} = r \cos\left[\phi\right] \delta\lambda \widehat{\lambda} + r \delta\phi\widehat{ \phi} + \delta r\widehat{r}. \end{equation}
We know from the definition of arc length $S$ that we can relate the change in the angle, let's call it $ \omega$, to the radius, $r$, via: \begin{equation} \omega = \frac{S}{r}. \end{equation} So, then for the $y$-coordinate, we find: \begin{equation} \delta \phi = \frac{\delta y}{r} \Rightarrow \delta y = r \delta \phi. \end{equation} Similarly for the $x$-coordinate, and noting that the radius is modulated $\cos\left[\phi\right]$, we get: \begin{equation} \delta \lambda = \frac{\delta x}{r \cos\left[\phi\right]} \Rightarrow \delta x = r \cos\left[\phi\right] \delta \lambda. \end{equation} Lastly, we note that $z$ and $r$ are simply translations by the Earth's radius $a$ via $z = r - a$, therefore: \begin{equation} \delta z = \delta r. \end{equation} And we note that we can extend these definitions to include the derivatives, as seen later, for Cartesian derivatives of the form: \begin{equation} (u, v, w)\big|_{\text{Cartesian}} = \left(\frac{\textrm{D}x}{\textrm{D}t}, \frac{\textrm{D}y}{\textrm{D}t}, \frac{\textrm{D}z}{\textrm{D}t} \right) . \end{equation}
An important derivative definition is as follows:
We define the "Material Derivative" of a quantitity $c$ as: \begin{equation} \frac{\textrm{D}c}{\textrm{D}t} = \partial_tc + \underline{u}\cdot\nabla c, \end{equation} for $\partial_t$ defining the internal rate of change of $c$ and $\underline{u}\cdot \nabla $ defining the advection of $c$.
Let's define $c(x,y,z,t)$. Then, by the definition of the differential, we find that small variations in all four independent variables gives: \begin{equation} \delta c = \partial_tc\delta t + \partial_x c \delta x + \partial_yc\delta y + \partial_zc\delta z. \end{equation} Then, we can define the velocities as: \begin{equation} \delta x = u\delta t \quad,\quad \delta y = v \delta t \quad,\quad \delta z = w \delta t. \end{equation} After substitution, division by $\delta t$, and taking $\delta \to 0$, we get: \begin{equation} \partial_t c\big|_{\text{Euler}} = \frac{\textrm{D}c}{\textrm{D}t}, \end{equation} as defined above.
N.B.: We can find the Spherical Coordinates extension as: \begin{equation} \frac{\textrm{D}c}{\textrm{D}t} = \partial_t c + \frac{u}{r \cos\left[\phi\right]}\partial_\lambda c + \frac{v}{r}\partial_\phi c + w \partial_r c, \end{equation} where we have defined: \begin{equation} (u, v, w)\big|_{\text{Spherical}} = \left(r\cos\left[\phi\right]\frac{\textrm{D}\lambda}{\textrm{D}t}, r\frac{\textrm{D}\phi}{\textrm{D}t}, \frac{\textrm{D}r}{\textrm{D}t} \right). \end{equation}
Next, we introduce the steamfunction $\psi$, which is defined as follows.
We define the streamfunction $\psi$ as: \begin{equation} u = - \partial_y \psi \quad \text{ and }\quad v = \partial_x \psi. \end{equation}
We can determine the timescales associated with overturning circulation described by $\psi$ as follows: \begin{equation} \tau = \frac{\text{volume}}{\text{volume flux}}. \end{equation} We can then imagine taking a slice orthoganol to which ever direction we care to determine the average flux through to get typical scales, e.g.: \begin{equation} \overline{v} \sim \frac{\text{volume flux}}{\text{depth}\times \text{width}} \quad \text{ or }\quad \overline{w} \sim \frac{\text{volume flux}}{\text{width} \times \text{length}}. \end{equation}
